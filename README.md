# MangaLibDownoloader

Многопоточный загрузчик манги с сайта MangaLib, который обходит защиту Cloudflare благодаря использованию реального Chromium-браузера (через `DrissionPage`). Программа собирает список глав, позволяет выбрать нужный диапазон для скачивания и быстро выкачивает страницы с удобным отображением прогресса и ETA.

## Особенности
* **Обход Cloudflare (Rate Limits 429)**: Имитация настоящего браузера и передача `cookies` в `requests.Session` для беспрепятственного скачивания.
* **Многопоточность**: Быстрое скачивание страниц с использованием пула потоков.
* **Гибкий выбор глав**: Можно скачивать как одну главу, так и целый диапазон (например `0-10`, `0,1,5`).
* **Удобный интерфейс**: Красивые индикаторы прогресса (Progress bars) на базе библиотеки `rich`.
* **Авто-повтор**: Работает система retry-очереди, которая вернется к пропущенным из-за лимитов главам и попробует выкачать их заново.

## Требования
* Python 3.8+
* Google Chrome (для работы `DrissionPage`)

## Установка

1. Склонируйте репозиторий или скачайте файлы в любую папку.
2. Установите необходимые библиотеки с помощью команды:
```bash
pip install -r requirements.txt
```

## Использование

Запустите главный скрипт:
```bash
python main.py
```

1. Скрипт запросит ссылку на тайтл MangaLib (например: `https://mangalib.org/ru/manga/otonari-no-tenshi-sama`).
2. Программа спарсит полный список переведенных глав.
3. Введите нужный диапазон в формате `НАЧАЛО-КОНЕЦ` (например: `0-20`) или выберите конкретные главы через запятую (например: `5, 7, 10`).
4. Остальное скрипт сделает сам! Скачанные тома и главы будут отсортированы и заботливо сложены в папку `downloads/` рядом со скриптом.

## Возможные проблемы
- **Ошибка 429 (Too Many Requests)**: Программа разработана для автоматического обхода этой ошибки через паузы (`random sleep`) и повторные попытки. Если ошибка persiste, скрипт добавит главу в очередь "повтора" и скачает ее в самом конце.
